Encryption Ransom

Cybersecurity 2019 Digital Forensics Coursework

Overview

Challenge

The challenge was to create a secure, educational demonstration of ransomware mechanics for cybersecurity education while ensuring the project remained ethical and could not be repurposed for malicious intent. Additionally, we needed to integrate with cloud storage for secure data handling.

Approach

  • Integrated Google Drive API for secure cloud storage of encryption keys

  • Built safeguards to ensure the project could only function in a controlled environment

Results

  • Created a fully functional educational demonstration tool
  • Successfully integrated with Google Drive for secure key management
  • Developed a user-friendly interface that explained the encryption process
  • Received excellent feedback from instructors for the technical implementation
  • Gained practical experience in ethical hacking and data encryption
  • Demonstrated strong problem-solving skills in a complex programming environment

Diabetes Prediction Using Machine Learning

Data Science 2021 Healthcare Research

Overview

Developed a project focused on creating a robust and highly accurate diabetes prediction model using machine learning techniques. The model analyzes various health parameters to predict diabetes risk, helping in early detection and prevention.

Challenge

Early diabetes detection remains a significant healthcare challenge, with many cases going undiagnosed until complications arise. Traditional screening methods can be time-consuming and expensive, limiting widespread implementation. The challenge was to develop an accurate, accessible prediction model that could assist healthcare providers in identifying at-risk patients.

Approach

  • Collected and preprocessed a comprehensive dataset of health parameters
  • Performed exploratory data analysis to identify key factors influencing diabetes risk
  • Implemented feature selection and engineering to optimize model inputs
  • Developed and trained multiple machine learning algorithms, including Random Forest, SVM, and Neural Networks
  • Conducted cross-validation to ensure model reliability and generalizability
  • Optimized hyperparameters to maximize prediction accuracy

Results

  • Achieved high prediction accuracy with the Random Forest algorithm outperforming other methods
  • Identified key health indicators with the strongest correlation to diabetes risk
  • Created a model with balanced precision and recall, minimizing both false positives and negatives
  • Developed visualizations to make model outputs interpretable for healthcare providers
  • Demonstrated the potential for ML-based screening to complement traditional diagnostic approaches
  • Provided a foundation for further research in predictive healthcare analytics

Technologies Used

Python Scikit-learn Pandas NumPy Matplotlib Random Forest
